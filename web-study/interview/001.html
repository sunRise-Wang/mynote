元素的水平垂直居中 ​

定位 + margin: auto ​

html
<html>
    <title></title>
    <body>
        <div class='father'>
         <div class='son'></div>
        </div>
    </body>
</html>
css
.father{
    position: relative;   //父元素相对定位
    width: 200px;
    height: 200px;
    border: 1px solid #f40
}

.father .son{
    position: absolute;    //子元素绝对定位
    top: 0;
    right: 0;
    bottom: 0;
    left: 0;                //撑满了整个父级
    margin: auto;   //实现水平垂直居中
    width: 50px;
    height: 50px;
    background-color: #8df
}

定位 + margin：负值 ​

html
<html>
    <title></title>
    <body>
        <div class='father'>
         <div class='son'></div>
        </div>
    </body>
</html>
css
.father{
    position: relative;   //父元素相对定位
    width: 200px;
    height: 200px;
    border: 1px solid #f40
}

.father .son{
    position: absolute;    //子元素绝对定位
    top: 50%;
    left: 50%;
    margin-left: -25px;
    margin-top: -25px;
    width: 50px;
    height: 50px;
    background-color: #8df
}

整个实现思路如下图所示：

  • 初始位置为方块1的位置
  • 当设置left、top为50%的时候，内部子元素为方块2的位置
  • 设置margin为负数时，使内部子元素到方块3的位置，即中间位置

这种方法不要求父元素的高度，也就是即使父元素的高度变化了，仍然可以保持在父元素的垂直居中位置

但是该方案需要知道子元素自身的宽高

定位 + transform ​

html
<html>
    <title></title>
    <body>
        <div class='father'>
         <div class='son'></div>
        </div>
    </body>
</html>
css
.father{
    position: relative;   //父元素相对定位
    width: 200px;
    height: 200px;
    border: 1px solid #f40
}

.father .son{
    position: absolute;    //子元素绝对定位
    top: 50%;
    left: 50%;
    transform: translate(-50%, -50%);   //将元素位移自己宽度和高度的-50%
    width: 50px;
    height: 50px;
    background-color: #8df
}

可以说这种方法是margin负值的替代方案，并不需要知道自身元素的宽高

flex弹性布局 ​

html
<html>
    <title></title>
    <body>
        <div class='father'>
         <div class='son'></div>
        </div>
    </body>
</html>
css
.father{
    display: flex;
    justify-content: center;
    align-items: center;
    width: 200px;
    height: 200px;
    background-color: #f40;
}
.father .son{
    width: 50px;
    height: 50px;
    background-color: #8df
}

flex布局是实现水平垂直居中的最简单最快的方法

display: flex时，表示该容器内部的元素将按照flex进行布局

align-items: center 表示这些元素将相对于本容器水平居中

justify-content: center 也是同样的道理垂直居中

grid网格布局 ​

html
<html>
    <title></title>
    <body>
        <div class='father'>
         <div class='son'></div>
        </div>
    </body>
</html>
css
.father{
    display: grid;
    align-items:center;
    justify-content: center;
    width: 200px;
    height: 200px;
    background-color: #f40
}
.father .son{
    width: 50px;
    height: 50px;
    background-color: #8df
}

grid网格布局和flex布局是一样的简单

table布局 ​

html
<html>
    <title></title>
    <body>
        <div class='father'>
         <div class='son'></div>
        </div>
    </body>
</html>
css
.father{
     display: table-cell;
     
     vertical-align: middle;
     text-align: center;
    
     width: 200px;
     height: 200px;
     background-color: #f40;
}
.father .son{
    display: inline-block;
    width: 50px;
    height: 50px;
    background-color: #8df
}

设置父元素为display: table-cell，子元素设置 display: inline-block。

利用vertical-align和text-align可以让所有的行内块级元素水平垂直居中

总结 ​

上述方法中，不知道元素宽高大小仍能实现水平垂直居中的方法有：

  • 利用定位 + margin:auto
  • 利用定位 + transform
  • flex布局
  • grid布局

根据元素标签的性质，可以分为：

  • 行内元素居中布局
  • 块级元素居中布局

元素居中布局 ​

水平居中

  • 行内元素可设置：text-align: center
  • flex布局设置父元素：display: flex; justify-content: center

垂直居中

  • 单行文本父元素确认高度：height === line-height
  • 多行文本父元素确认高度：display: table-cell; vertical-align: middle

块级元素居中布局 ​

水平居中

  • 定宽: margin: 0 auto
  • 绝对定位+left:50% + margin:负自身一半

垂直居中

  • position: absolute设置 left、top、margin-left、margin-top(定高)
  • display: table-cell
  • transform: translate(x, y)
  • flex(不定高，不定宽)
  • grid(不定高，不定宽)，兼容性相对比较差
