html4 ​

开始之前你要明白

html4 是 html5标准还没有发布之前所有的标准

1.什么是html ​

1.1-定义 ​

超文本标记语言（英语：HyperText Markup Language，简称：HTML）

是一种用于创建网页的标准标记语言。

1.2-html 的基本骨架 ​

html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>我是第一个html文件</title>
  </head>
  <body>
    <h1>我的第一个标题</h1>
    <p>我的第一个段落。</p>
  </body>
</html>

1.3-后缀名 ​

.html

.htm

以上两个后缀名没有区别，均可使用

1.4-总结 ​

HTML 不是一种编程语言，而是一种标记语言

标记语言 是一套标记标签

HTML 使用标记标签来描述网页

2.Html 元素 ​

html元素

指的是从 [开放标签] 到 [结束标签] 的所有代码

3.Html属性 ​

属性

是为元素提供附属信息的

html标签 是可以书写属性的

但是属性是要写在开放标签上的

属性的写法，主要都是 以 “属性名='属性值'” 的形式书写

txt
有一些属性是所有 html标签 都通用的
例如：

属性        属性值                作用
title       text              规定元素的额外信息（可在工具提示中显示）
class       classname         规定元素的类名（classname）
id          id                规定元素的唯一 id
style       style_definition  规定元素的行内样式（inline style）
