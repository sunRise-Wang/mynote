html4 ​

开始之前你要明白

html4 是 html5标准还没有发布之前所有的标准的统称（自己的理解）

在这里有些代码用到了html5标准的东西，因为html5已经是一个经典的版本

1.什么是html ​

1.1-定义 ​

超文本标记语言（英语：HyperText Markup Language，简称：HTML）

是一种用于创建网页的标准标记语言。

1.2-html 的基本骨架 ​

html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>我是第一个html文件</title>
  </head>
  <body>
    <h1>我的第一个标题</h1>
    <p>我的第一个段落。</p>
  </body>
</html>

1.3-后缀名 ​

.html

.htm

以上两个后缀名没有区别，均可使用

1.4-总结 ​

HTML 不是一种编程语言，而是一种标记语言

标记语言 是一套标记标签

HTML 使用标记标签来描述网页

2.Html元素 ​

html元素

指的是从 [开放标签] 到 [结束标签] 的所有代码

大多数 HTML元素 可以嵌套（HTML元素可以包含其他 HTML元素）

HTML文档 由相互嵌套的 HTML元素构成

3.Html属性 ​

属性

是为元素提供附属信息的

html标签 是可以书写属性的

但是属性是要写在开放标签上的

属性的写法，主要都是 以 “属性名='属性值'” 的形式书写 有一些属性是所有 html标签 都通用的 例如：

属性属性值作用
titletext规定元素的额外信息（可在工具提示中显示）
classclassname规定元素的类名（classname）
idid规定元素的唯一 id
stylestyle_definition规定元素的行内样式

注意

其它的属性都是其它标签独有的，也有一样的，而上面的四种属性都是通用的

所以等看到其它标签的时候再仔细研究这些属性

4.标题标签[h1~h6] ​

分类:

· 它属于排版标签

· 双标签

· 块级元素

标题(header)

标题标签 有六种 分别有 h1~h6

从定义上讲，h1是最大的标题，h6是最小的

html
<h1>一级标题</h1> //最大的标题
<h2>二级标题</h2>
<h3>三级标题</h3>
<h4>四级标题</h4>
<h5>五级标题</h5>
<h6>六级标题</h6> //最小的标题

注意

标题标签只能用于标题，不要仅仅为了生成粗体或大号字体而使用标题标签

标题标签很重要，因为用户可以通过标题快速的浏览网页

应该将 h1 用作主标题（最重要的），其后是 h2（次重要的），再其次是 h3，以此类推。

5.段落标签[p] ​

分类:

· 它属于排版标签

· 双标签

· 块级元素

段落(paragraph)

段落标签 p 可以把html文档分成一个个段落

html
<p>这是一个段落标签</p>

6.链接标签[a] ​

分类:

· 它属于链接标签

· 双标签

· 行内元素

超链接(链接)

使用标签 a 来设置超文本链接

超链接可以是一个字、一个词、一张图......

点击它可以跳转到新的文档或者是当前文档的某个地方

可以通过 a标签的特有属性 href来指定要跳转的链接地址

html
<a id='html文档书签' href='要跳转的地址' target='_blank'>这是一个超链接</a>
id属性： 这里的id属性是可以配合其它的超链接实现定位
href属性：填写要跳转的地址
target属性: 定义被链接的文档在何处显示 _blank是在新的标签页打开 默认是在本身标签上打开
download属性：跳转文件时下载

在默认情况下，a标签具有以下的特点:

鼠标悬停在链接上会变成小手

  • 一个未访问过的链接显示为蓝色字体并带有下划线。
  • 访问过的链接显示为紫色并带有下划线。
  • 点击链接时，链接显示为红色并带有下划线

注意

这里的默认样式是可以通过后面的css样式更改

a标签不能嵌套a标签

7.图像标签[img] ​

分类:

· 它属于图像标签

· 行内块元素

· 单标签

图像标签

这个标签可以在页面上显示图片

img 是空标签，他没有结束标签

它是靠属性来操作标签

html
<img src='图片的链接' alt='图片失效后所呈现的文字' width='图片的宽度' height='图片的高度'/>
  • 设置图片的大小是个好习惯，如果不设置就会影响页面的布局
  • 如果页面加载的图片过多，不能一次性全都加载出来，因为加载图片需要时间，所以需要图片懒加载
  • 图片链接一定要写对，不然会显示裂开的图片

8.Html文档[head标签] ​

head标签

它包含了所有头部标签元素

html
<head>
 <meta /> 网页的元信息
 <link /> 定义文档与外部资源的联系
 <title>网页的标题</title>
    <style></style>  //内联的样式表
 <script></script> //js脚本
    <base /> //基本的链接地址/链接目标
</head>

8.1-title标签 ​

title标签

定义网页的标题

这是文档必需的

  • 浏览器工具栏的标题
  • 收藏网页时，收藏夹中的标题
  • 搜索引擎结果页面的标题

8.2-link标签 ​

link标签

定义文档与外部资源之间的联系

通常链接到外部的样式文件

8.3-style标签 ​

style标签

可以直接添加样式

8.4-script标签 ​

script标签

用于加载脚本文件

也可以在body中元素之后加载js文件

因为先让浏览器加载html结构和css样式，最后再加载js脚本，这样效果会很好

8.5-meta标签 ​

meta标签

定义网页的元信息，它不会显示在网页中，但会被浏览器解析

通常用于指定网页的描述，关键词，文件的最后修改时间，作者，和其他元数据

html
搜索引擎的关键词
<meta name='keywords' content='内容'/>
网页的描述内容
<meta name='description' content='内容'/>
定义网页的作者
<meta name='author' content='内容'/>
每30秒钟刷新当前页面
<meta http-equiv="refresh" content="30">
......
需要了解的，可以去mdn去查看

9.列表标签[ul,ol,dl] ​

9.1-无序列表标签[ul] ​

无序列表[ul]

一个项目的列表，此列项目使用粗体圆点（典型的小黑圆圈）进行标记

html
<ul>
 <li>我是无序列表里的一项</li>
    <li>我是无序列表里的一项</li>
    <li>我是无序列表里的一项</li>
</ul>

页面呈现的结果:

  • 我是无序列表里的一项
  • 我是无序列表里的一项
  • 我是无序列表里的一项

9.2-有序列表标签[ol] ​

有序列表[ol]

一个项目的列表，此列项目使用 数字 进行标记

html
<ol>
   <li>我是有序列表里的一项</li>
    <li>我是有序列表里的一项</li>
    <li>我是有序列表里的一项</li>
</ol>

页面呈现的结果:

  1. 我是有序列表里的一项
  2. 我是有序列表里的一项
  3. 我是有序列表里的一项

9.3-自定义列表标签[dl] ​

自定义列表[dl]

自定义列表不仅仅是一列项目，而是项目及其注释的组合。

自定义列表以 dl 标签开始。每个自定义列表项以 dt 开始。每个自定义列表项的定义以 dd 开始。

html
<dl>
    <dt>我是名词</dt>
    <dd>我来解释</dd>
</dl>

页面呈现的结果:

我是名词

— 我来解释

10.表格标签[table] ​

表格标签[table]

table标签可以用来创建表格

每个表格都有若干行，使用 tr 标签来定义

每行都被分割成若干个单元格，使用 td 标签来定义

td 是table data 的缩写，数据单元格的内容

数据单元格可以包含文本、图片、列表、段落、表单、水平线、表格等等

html
<table>
    <caption>表格的表头</caption>
    表格头部
    <thead>
        <tr>
         <th></th>
            <th></th>
        </tr>
    </thead>
    表格主体
    <tbody>
     <tr>
         <td></td>
         <td></td>
     </tr>
        <tr>
         <td></td>
         <td></td>
     </tr>
    </tbody>
    表格脚注
    <tfoot>
        <tr>
            <td></td>
            <td></td>
        </tr>
    </tfoot>
</table>

属性值

border: 控制是否显示边框

cellspacing: 控制单元格之间的间隔

width: 表格或单元格的宽度

height: 表格或单元格的高度

align: 控制单元格内的文字的水平位置 (left、center、right)

valign: 控制单元格内的文字的垂直位置 (top、middle、bottom)

colspan: 跨列合并单元格

rowspan: 跨行合并单元格

11.表单标签[form] ​

表单标签[form]

form标签用来收集用户的输入信息

文档中有一块区域，里面有表单控件，收集用户输入的信息，并提交到对应的web服务器上

html
基本骨架
<form action='目标服务器的地址' method='get & post' target='打开页面的方式'>
    表单控件:
    input控件
    文本域
    <input type='text' name='存储表单数据的变量名'/>
    密码控件
    <input type='password' name='存储表单数据的变量名'>
    单选按钮
    <input type='radio' name='存储表单数据的变量名' value='提交表单时的结果'>
    复选框
    <input type='checkbox' name='存储表单数据的变量名' value='提交表单时的结果'>
    提交按钮 | 重置按钮 | 普通按钮
    <input type='submit&reset' value='提交按钮的文字内容'>
    
    textarea控件
    <textarea cols='列的长度' rows='行的长度'></textarea>
    
    下拉框控件
    <select name='存储表单数据的变量名'>
        <option selected value='提交表单时的结果'>项目1</option>
        <option value='提交表单时的结果'>项目2</option>
    </select>
    
    <label for='与表单控件的的id一致'></label>
    它可与表单控件相关联，关联之后点击文字，与之对应的表单控件会获取焦点
    表单控件需要嵌套在 label标签中
</form>

12.Html区块元素 ​

html 可以通过 div标签 和 span标签 将元素组合起来

区块元素

大部分html元素可以被分类成 块级元素和行内元素

块级元素：独占一行

div 没有特定的含义

行内元素：不独占一行

span 没有特定的含义

13.Html字符实体 ​

字符实体

HTML 中的预留字符必须被替换为字符实体。

一些在键盘上找不到的字符也可以使用字符实体来替换。

html
空格    空格    &nbsp;
<      小于号   &lt;
>    大于号   &gt;
&      和号     &amp;
￥     人民币    &yen;
©      版权     &copy;
®      注册商标  &reg;
™      商标     &trade;

注意

虽然 html 不区分大小写，但实体字符对大小写敏感
